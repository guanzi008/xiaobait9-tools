# 小白T9工具箱 (Xiaobai T9 Tools)

[![License](https://img.shields.io/badge/license-GPLv3-blue.svg)](LICENSE)
[![Platform](https://img.shields.io/badge/platform-linux--deepin-lightgrey.svg)]()

小白T9工具箱是一款专为Deepin/UOS系统设计的输入法配置工具，主要用于安装和配置中州韵输入法引擎(Rime)及小白九键拼音输入方案。

## 项目地址

- Gitee: https://gitee.com/yanxi-bamboo-forest/xiaobait9-tools
- GitHub: https://github.com/YXZL985/xiaobait9-tools

## 功能特性

- 一键安装中州韵输入法引擎(Fcitx5-Rime)
- 自动部署小白九键拼音输入方案
- 支持简体和繁体拼音输入
- 图形化界面操作，简单易用
- 一键重新部署输入法引擎

## 安装依赖

- Deepin/UOS 操作系统
- Qt5
- DTK (Deepin Toolkit)
- Fcitx5
