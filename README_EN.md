# Xiaobai T9 Tools

[![License](https://img.shields.io/badge/license-GPLv3-blue.svg)](LICENSE)
[![Platform](https://img.shields.io/badge/platform-linux--deepin-lightgrey.svg)]()

Xiaobai T9 Tools is an input method configuration tool designed specifically for Deepin/UOS systems, primarily used for installing and configuring the Rime input method engine and the Xiaobai T9 Pinyin input scheme.

## Repository

- Gitee: https://gitee.com/yanxi-bamboo-forest/xiaobait9-tools
- GitHub: https://github.com/YXZL985/xiaobait9-tools

## Features

- One-click installation of Rime input method engine (Fcitx5-Rime)
- Automatic deployment of Xiaobai T9 Pinyin input scheme
- Support for Simplified and Traditional Chinese input
- Graphical user interface, easy to use
- One-click redeployment of input method engine

## Dependencies

- Deepin/UOS Operating System
- Qt5
- DTK (Deepin Toolkit)
- Fcitx5
